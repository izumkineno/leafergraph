# Titlebar

## 作用
- 顶部品牌区、文档摘要、命令工具栏与运行控制按钮。

## 导出
- `EditorTitlebarView`
- `EditorTitlebarConnected`
- `EditorTitlebarViewProps`

## 依赖
- 连接态版本依赖 `EditorProvider` 上下文。

## 使用方式
```tsx
import { EditorTitlebarConnected } from "leafergraph-editor/ui/titlebar";
import "leafergraph-editor/ui/titlebar/styles.css";
```
