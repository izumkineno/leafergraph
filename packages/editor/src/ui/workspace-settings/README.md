# Workspace Settings

## 作用
- 承接扩展、authority、偏好、快捷键等设置面板。

## 导出
- `EditorWorkspaceSettingsView`
- `EditorWorkspaceSettingsConnected`
- `EditorWorkspaceSettingsViewProps`

## 使用方式
```tsx
import { EditorWorkspaceSettingsConnected } from "leafergraph-editor/ui/workspace-settings";
import "leafergraph-editor/ui/workspace-settings/styles.css";
```
