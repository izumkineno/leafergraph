# Node Library Preview

## 作用
- 节点库 hover/focus 的浮层预览。

## 导出
- `NodeLibraryHoverPreviewOverlay`（View）
- `EditorNodeLibraryPreviewConnected`
- 预览辅助函数与类型（`helpers`）

## 使用方式
```tsx
import { EditorNodeLibraryPreviewConnected } from "leafergraph-editor/ui/node-library-preview";
import "leafergraph-editor/ui/node-library-preview/styles.css";
```
