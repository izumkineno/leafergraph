# Dialog

## 作用
- 提供 editor 通用对话框壳层。
- 统一焦点返回、`ESC` 关闭、遮罩点击关闭行为。

## 导出
- `AppDialog`（View）
- `AppDialogConnected`（当前等价于 View）
- `AppDialogProps`
- `AppDialogSize`

## 使用方式
```tsx
import { AppDialog } from "leafergraph-editor/ui/foundation";
import "leafergraph-editor/ui/foundation/dialog/styles.css";
```
